# 更新日志 (Changelog)

## [v1.1.0] - 2026-04-21

### 简体中文 (Chinese)
#### **新增内容**
- **方块与物品**：新增了“原矿块”与“矿物块”及其对应的基础属性。
- **视觉资源**：为原矿块和矿物块添加了全新的材质图片。
- **合成与熔炼**：
    - 添加了基础合成表（Recipe）。
    - 实现了燃料燃烧逻辑，部分新方块现在可作为燃料或进行烧炼。

#### **优化与修复**
- 优化了项目文档结构，为后续功能展示做准备。

---

### English
#### **Added**
- **Blocks & Items**: Added "Raw Ore Block" and "Mineral Block" with their respective base properties.
- **Visual Assets**: Added new textures and sprites for both Raw Ore and Mineral Blocks.
- **Crafting & Smelting**:
    - Implemented basic crafting recipes.
    - Added fuel and burning logic; new blocks can now be used as fuel or smelted.

#### **Improvements**
- Refined project documentation structure for future feature showcases.
