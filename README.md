## 简体中文 (Chinese)

### **项目简介**
这是一个基于 **Fabric API** 开发的 Minecraft 基础功能模组。本项目目前主要用于**个人学习、技术测试及基础功能演示**，旨在探索模组开发的底层逻辑与实现方式。

### **当前状态**
⚠️ **注意**：本模组目前处于早期测试/实验阶段（Experimental/Testing），仅建议用于开发环境测试，请勿直接用于重要的生存存档。
